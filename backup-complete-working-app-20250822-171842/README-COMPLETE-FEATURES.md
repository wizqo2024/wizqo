## 🔧 **SETUP & DEPLOYMENT**

### **Environment Variables Required:**
```bash
OPENROUTER_API_KEY=your_openrouter_key
YOUTUBE_API_KEY=your_youtube_key
SUPABASE_URL=your_supabase_url
```
